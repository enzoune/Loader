## 🌀 CSS Loading Animation (Optimized Variables)
> Um componente de carregamento (Loader), construído com HTML5 e CSS3 moderno, utilizando variáveis personalizadas para animações cíclicas sincronizadas.

## 📖 Sobre o Projeto
Este projeto faz parte do meu portfólio de estudos em Programação Full-Stack. O objetivo principal foi desenvolver um componente de feedback visual que seja leve e independente de bibliotecas externas ou JavaScript.

O loader utiliza uma estrutura de 15 elementos circulares (<span>) que, através de cálculos matemáticos no CSS, criam um efeito visual de movimento infinito. Este tipo de componente é vital para melhorar a percepção de tempo de carregamento em aplicações web reais.

## ✨ Destaques e Aprendizados Técnicos
* **CSS Custom Properties (Variáveis):** Uso estratégico de variáveis inline (--i) para passar índices diretamente do HTML para o CSS.

* **Animação Sequencial:** Implementação de atrasos de animação (animation-delay) calculados dinamicamente com base nas variáveis, evitando a escrita de 15 regras manuais diferentes

* **Código Limpo:** Estrutura HTML minimalista e semântica dentro de um container dedicado (.dots).

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação dos elementos da animação.

* **CSS3:** Gerenciamento de keyframes, variáveis personalizadas e estilização de layout.
